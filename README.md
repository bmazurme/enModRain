# enModRain

### Калькулятор 
* расчета дождевого стока с кровли здания
* диаметр отверстия шайбы 
* расход тепла для нагрева горячей воды

#### Используемые технологии
* HTML
* CSS
* JS
* БЭМ
* KaTeX
