# enModRain

### Калькулятор 
* расчета дождевого стока с кровли здания
* диаметра отверстий шайб 
* расхода тепла для нагрева горячей воды
* циркуляционного расхода горячей воды
* регулирующего объема бака-аккумулятора

#### Используемые технологии
* HTML
* CSS
* JS
* БЭМ
* KaTeX
